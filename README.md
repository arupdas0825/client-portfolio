## Usage

- Fork/clone the repo and customize the content to match your personal brand or professional work.
- Update the information in HTML/JS files as desired.
- Deploy to GitHub Pages, Vercel, Netlify, or your preferred hosting service.

## Customization

Want to personalize your portfolio?  
- Replace files in the `assets/` directory with your own images/logos.
- Update the content of `index.html` and JavaScript files in `js/` as needed.
- Modify the styles in `css/` to change colors, fonts, or layout.
